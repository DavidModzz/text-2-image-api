Text2ImageApi

Uma api Open-Source para criação de imagens a partir de emojis.

Como usar

Para usar, você pode estar clonando o código disponivel do meu Github ou por meio dos endpoints.

  • Imagem estática

    /ttp?texto=texto&cor=3f9cd6

    Passe como parametros [Texto] e [Cor] para gerar uma imagem com o texto e cor selecionados.

  • Imagem animada webp only

    /attp?texto=texto

    Passe como parametro [Texto] para gerar um video com o texto selecionados.
